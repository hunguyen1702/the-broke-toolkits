# New Spec

Guide the user through planning a new feature implementation using systematic discovery, approach analysis, and structured task breakdown.

This produces 3 artifacts in `docs/spec/{slug}/`: a discovery report, an analysis report, and a detailed implementation spec with dependency-ordered tasks.

If `$ARGUMENTS` is provided, use it as the initial feature description and pass it to Step 1.
