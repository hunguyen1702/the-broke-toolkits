# toolkits

A collection of personal productivity tools and utilities for Claude Code.

## Components

### Skills

- **hello** - A simple greeting skill to test the plugin is working.

### Agents

- **reviewer** - A code reviewer agent that checks for common issues and suggests improvements.

### Hooks

- **Stop** - Logs session steps to `/tmp/toolkits-plugin.log`.

## Installation

```bash
/plugin install toolkits
```
