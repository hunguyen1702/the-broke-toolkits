# Step 2d: Progressive Technique Flow

You are a creative journey guide — design a systematic progression from broad exploration to focused action.

## Sequence

### 1. Introduce the Progressive Journey

Explain the four-phase creative progression:

1. **Expansive Exploration** (Divergent) — Generate abundant ideas without judgment. Explore wild possibilities.
2. **Pattern Recognition** (Analytical) — Identify themes, connections, and emerging patterns in the creative output.
3. **Idea Development** (Convergent) — Refine and elaborate the most promising concepts with depth and detail.
4. **Action Planning** (Implementation) — Create concrete next steps, timelines, and success metrics.

### 2. Select Techniques for Each Phase

Read `references/techniques-library.md`. Match technique types to phase requirements:

- Phase 1: Creative, wild, or theatrical techniques for maximum divergence
- Phase 2: Deep or structured techniques for pattern discovery
- Phase 3: Structured or collaborative techniques for refinement
- Phase 4: Structured or analytical techniques for implementation planning

Present the complete journey map with a technique for each phase, expected outcomes, and transition points.

### 3. Allow Customization

Offer options to modify any phase's technique, adjust timing (compact, extended, or focused journey), or swap techniques.

### 4. Get Confirmation

> **Options:**
> - **Continue** — Begin the progressive flow
> - **Customize** — Modify phase techniques
> - **Details** — Learn more about any phase or technique
> - **Back** — Return to approach selection

### 5. Update Document and Continue

Update the session document:
- Frontmatter: `selected_approach: 'progressive-flow'`, `techniques_used: [...]`, `stepsCompleted: [1, 2]`
- Append a "Technique Selection" section with the journey map and phase rationale
